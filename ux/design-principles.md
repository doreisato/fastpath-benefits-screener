# Mobile-First Design Principles

1. **Constraint Drives Focus**: Small screens force ruthless prioritization. Only one primary action per screen.
2. **Touch Targets**: Minimum 44x44 CSS pixels for all buttons and interactive elements. No precision tapping required.
3. **Vertical Rhythm**: Scroll is natural. Horizontal swipes are friction. Stack content vertically.
4. **Typography Scale**: Base font size 16px minimum to prevent auto-zoom on iOS inputs. Keep it readable.
5. **Performance is UX**: Under 2-second Time to Interactive (TTI) on 3G networks.
